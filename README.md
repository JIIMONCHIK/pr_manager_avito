# PR Reviewer Assignment Service

Сервис для автоматического назначения ревьюеров на Pull Request'ы.

## Запуск

```bash
make run
```

## Архитектура
- FastAPI - веб-фреймворк

- PostgreSQL - база данных

- SQLAlchemy - ORM

- Docker - контейнеризация

## API Документация
- Swagger UI: http://localhost:8080/docs
